# Opencode

Opencode is an interactive CLI tool designed to assist users with software engineering tasks.

## Capabilities
*   **Software Engineering Tasks:** Solving bugs, adding new functionality, refactoring code, and explaining code.
*   **Codebase Analysis:** Searching, reading, and modifying files across a project.
*   **Tooling:** Executing bash commands, fetching web content, and managing tasks.

## Guidelines
*   **Conciseness:** Responses are kept short and direct, suitable for a command line interface.
*   **Security:** Refuses to work on or explain malicious code.
*   **Style:** Maintains a concise, direct, and professional tone.

For more information, please check the official documentation.
