# OpenCode: The Open Source AI Coding Agent

OpenCode is a powerful, open-source AI coding agent designed to assist developers directly within their workflow, whether in a terminal, IDE, or dedicated desktop application.

## 💡 Core Functionality
OpenCode acts as an intelligent coding partner, providing assistance across various software engineering tasks. It is designed to be highly flexible and integrated:

*   **Multi-Platform Support:** Available as a terminal interface, desktop app, and IDE extension.
*   **LSP Enabled:** Automatically loads the right Language Server Protocols (LSPs) for enhanced LLM capabilities.
*   **Multi-Session:** Allows users to run multiple agents in parallel on the same project.
*   **Model Agnostic:** Supports over 75 LLM providers through Models.dev, including local models, Claude, GPT, Gemini, and others.

## 🛡️ Key Features & Design Principles
*   **Privacy First:** OpenCode does not store any of your code or context data, making it suitable for privacy-sensitive environments.
*   **Community Driven:** Trusted by millions of developers, boasting a large community of contributors and a massive GitHub presence.
*   **Integrations:** Supports logging in with popular services like GitHub Copilot and ChatGPT Plus/Pro.
